# CUJ

Run-time program generator in C++

## Requirements

* LLVM 11.1.0 and (optional) CUDA 10 (other versions may work but haven't been tested)
* A C++17 compiler

## Usage

### Getting Started

```
git clone https://github.com/AirGuanZ/cuj.git
cd cuj
mkdir build
cd build
cmake -DLLVM_DIR="your_llvm_dir" ..
```
